Log all requests to your API

How do you log all requests made to your API?

Steps

  • Create new handler which logs all requests
  • Make sure the logger won't break on an exception
  • Retrieve ipaddress harder than it seems
  • Log user credentials when authenticated
  • Reconstruct the original request and response
