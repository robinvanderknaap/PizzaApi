Pizza API

The Pizza API is an example project to demonstrate the most important technical aspects of setting up an API. It's not about REST, or how one should design its public API, but covers the main technical building blocks used when setting up a WebApi project.

These pages are used as a narrative for a presentation. Each page contains links to the relevant code portions in the solution, this should make it easier to find what you are looking for.

Source code

The source code of the Pizza API is hosted on GitHub. Clone or download the source, open solution file with a recent version of Visual Studio and hit F5.

Use an API client like DHC (Chrome Extension) to consume the API.
